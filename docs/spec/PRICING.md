# AI 视频生成模型定价说明

本文档说明了 VideoFly 平台 AI 视频生成模型的定价策略。

---

## 模型定价详情

### Sora 2 (OpenAI)

详情页：https://evolink.ai/zh/sora-2

| 时长 | 无水印  | 有水印  |
| ---- | ------- | ------- |
| 10s  | 10 积分 | 16 积分 |
| 15s  | 20 积分 | 24 积分 |

**规格**:

- 支持时长: 10s, 15s
- 宽高比: 16:9, 9:16
- 图片输入: 支持 (可选)

---

### Wan 2.6

详情页：https://evolink.ai/zh/wan-2-6

| 分辨率 | 5s       | 10s      | 15s      |
| ------ | -------- | -------- | -------- |
| 720p   | 156 积分 | 312 积分 | 468 积分 |
| 1080p  | 260 积分 | 522 积分 | 782 积分 |

**规格**:

- 支持时长: 5s, 10s, 15s
- 宽高比: 16:9, 9:16, 1:1, 4:3, 3:4
- 分辨率: 720p, 1080p
- 图片输入: 支持 (可选)
- 音频生成: 支持
- 视频参考: 支持

---

### Veo 3.1 (Google)

详情页，我们对应的是 Veo 3.1 Fast Litehttps://evolink.ai/zh/veo-3-1?model=veo-3-1-fast-lite

| 规格         | 积分     |
| ------------ | -------- |
| 720p / 1080p | 60 积分  |
| 4K           | 180 积分 |

**规格**:

- 支持时长: 4s, 6s, 8s
- 宽高比: 16:9, 9:16
- 分辨率: 720p, 1080p
- 图片输入: 最多 2 张
- 输出数量: 1-4 (2-4 需要 Pro)

---

### Seedance 1.5 Pro (按秒计费)

详情页：https://evolink.ai/zh/seedance-1-5-pro

| 分辨率 | 音频 | 4s       | 5s       | 6s       | 8s       | 10s      | 12s      |
| ------ | ---- | -------- | -------- | -------- | -------- | -------- | -------- |
| 480p   | 否   | 16 积分  | 20 积分  | 24 积分  | 32 积分  | 40 积分  | 48 积分  |
| 720p   | 否   | 36 积分  | 44 积分  | 54 积分  | 72 积分  | 90 积分  | 108 积分 |
| 720p   | 是   | 76 积分  | 94 积分  | 112 积分 | 150 积分 | 188 积分 | 226 积分 |
| 1080p  | 否   | 84 积分  | 104 积分 | 126 积分 | 168 积分 | 210 积分 | 252 积分 |
| 1080p  | 是   | 168 积分 | 210 积分 | 252 积分 | 336 积分 | 420 积分 | 504 积分 |

**规格**:

- 支持时长: 4s - 12s (任意整数)
- 宽高比: 16:9, 9:16, 1:1, 4:3, 3:4, 21:9
- 分辨率: 480p, 720p, 1080p
- 图片输入: 0-2 张
- 音频生成: 支持
